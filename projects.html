Caesar Cipher

Project to make a Caesar cipher that takes a user's input and encrypts it by taking the ascii code from each character and moving it in the alphabet by a specified amount. So, an input of A being moved 3 places would become D.

Scratch Project - Intro to Computer Science

This project was to make a simple game in Scratch as an introduction coding using Scratch's block-based visual programming language.


Check it out on Scratch!

Instructions

As a member of the Space Force you have to protect the Moon Base from the alien invasion. If a single alien gets past your line, you lose and are sent back to the start.

Notes and Credits

Sound Effects:
"Pew" - Made by recording myself saying the word pew into Audacity where noise was also reduced then exported as an mp3 using the LAME encoder. Mp3 file type was chosen over other formats such as WAV or FLAC because this project wouldn't benefit from a lossless compression and would benefit from a smaller file size that could be accomplished with mp3 which is heavily supported online. Audio was then softened using the built-in sound editor in Scratch after deciding it was too loud.
------------------------------------------------------------------------
Graphics:
Laser Blast (Blast2) - I created the laser blast using Inkscape so that it could be made as a svg vector file allowing it to be scaled up or down without a loss in quality. This graphic was created this was because I was unsure of what the final dimensions would be, and since vector files save the shape of the curves within the image instead of individual pixels, it could take on any dimensions needed without needing to be redone.
The graphic was made using the "Draw calligraphic or brush strokes tool" then later converted different vector layers using the trace bitmap tool.
All other Graphics - I drew the rest of the graphics in Photoshop due to my familiarity with the program. These were exported as PNG files, to allow the use of an alpha channel, but were a lower quality than vector files because of saving pixel color data instead of shapes. The easiest way to see the difference is when comparing the edge of the moon in the background which is highly pixelated and looks like a blocky circle, to the clean smooth edges on the laser blast.
------------------------------------------------------------------------
Operators:
AND – Used in the “Alien” sprite in the tree starting with “When I receive: An Alien was shot.” The and function is used to check if the “Blast2” sprite is touching the alien clone and making sure it is still within the gameplay window. This was done because the non-clone version of the alien moves to the position the clone spawns in and although it is hidden, it can still be shot and affect the hidden variable “Enemies Alive.” Since both conditions must be true to move forward in the loop, it prevents waves from being ended prematurely.
OR – Used in the “Blast2” sprite in the tree starting with “When I start as a clone.” The or function was used in the repeat until loop to specify two different cases where the loop could end. This would allow the blast to move across the screen to a certain X position, or until it touched one of the Alien sprite clones.
NOT – Used in the “Blast2” sprite in the tree starting with “When space key pressed.” This was used to prevent the “Blast2” sprite clones from being spawned when on the title screen before the game started. So, when the current wave is 0, which is the default state, it won’t allow them to spawn when space is pressed.
------------------------------------------------------------------------ Other required descriptions: Boolean Logic – Boolean logic is a type of mathematical logic that results in one of two outputs: True or False. All computers today run on Boolean logic where a low voltage results in a digital 0 signal for false and high voltage which results in a digital 1 for true. The actual inner workings of Boolean logic can be broken down into 4 operators: AND, OR, NOT, and XOR. These 4 operators combined in various patterns can create circuits including transistors and are the fundamental building blocks of modern-day computers.
Number Systems - Number systems are a method of writing numbers in different bases. Bases represent the base number that has a power attached to it. In a base 2 number system, also known as binary, 2 is the base number and all numbers in existence can be expressed by a variation of 2's carried to different powers and added together. Overall the world is familiar with a base 10 number system where all numbers are expressed as 10 carried to different powers and added together. Another common base is base 16, or hexadecimal. In hexadecimal, or hex for short, the base is 16 and all numbers are expressed as 16 carried to different powers. Since people are usually only familiar with the digits 0-9, hex had to get creative in order to represent higher numbers. In hex, decimal 10 is represented as A and decimal 15 is represented as F. The numbers in between take B, C, D, and E respectively.
